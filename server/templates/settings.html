Pi Device Setup

One-time setup — no monitor or keyboard needed

Step 1 — Flash the SD card
Use Raspberry Pi Imager. Before writing, click the OS Customisation button (gear icon) and configure:
✓ Hostname: lockin
✓ Username & password of your choice
✓ WiFi network name & password
✓ Enable SSH (under Services tab)
This handles WiFi and SSH automatically — you won't need to create wpa_supplicant.conf or an ssh file manually.
Step 2 — Add two files to the boot partition
After flashing, plug the SD card back in. The bootfs partition appears as a drive. Download and copy both files into it:
⬇ lockin.conf ⬇ lockin-firstrun.sh
Step 3 — Append the installer to Pi Imager's setup script
Pi Imager placed a firstrun.sh in the boot partition. Open it in a text editor and paste the line below before the final rm -f /boot/firstrun.sh line:
bash /boot/lockin-firstrun.sh
Step 4 — Power on
Insert the SD card and power on the Pi. It will connect to WiFi, install LockIn automatically, then reboot into the focus monitor — no SSH, no monitor, no keyboard ever needed. The first boot takes around 3–5 minutes.
